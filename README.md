<h2>Program walk-through:</h2>
Using GitHub plugins to connect Autodesk Revit with Speckle and Power BI, this project demonstrates how BIM models can be shared online and visualized through interactive dashboards. The workflow enables Revit models to be synchronized with Speckle and integrated into Power BI to display project data, including quantities and properties of structural elements such as beams, columns, slabs, and other building components. The dashboards provide clear tables, charts, and visual analytics that support project coordination, reporting, and data-driven decision-making.

Navisworks is an essential tool for identifying and preventing potential issues before construction begins. In this project, I used Navisworks to create construction simulations (4D) by linking a Revit model with a Microsoft Project schedule. I also performed clash detection to identify conflicts between building elements, such as pipes, walls, and structural components. This workflow helps detect design inconsistencies, improve coordination among disciplines, and reduce costly errors during the construction phase.

These software applications can be integrated into a BIM workflow to improve project coordination and structural design. For example, AutoCAD can be used to develop the initial layout of the project, which can then be imported into Autodesk Robot Structural Analysis to perform structural analysis, verify compliance with design codes, and evaluate the structure under different loading conditions. Finally, the analytical model can be coordinated with Autodesk Revit to produce a detailed BIM model, construction documentation, and project visualization. This workflow enhances collaboration, improves design accuracy, and streamlines the engineering process.
